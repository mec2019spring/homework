# 直流双闭环调速

在采用理想可调电流源的基础上，分别采用纯比例P控制和比例积分控制，
选择合适的比例增益（和积分增益），得到较为理想的阶跃响应曲线。
以下为该阶跃响应与仿真作业2的理想阶跃响应的差别的比较。

经过对比可知：双闭环调速电路基本没有静差，静态特性很硬，随负载增大（若ki足够大）转速几乎不变
动态响应快，启动时间短，调整方便，且没有超调，缺点在于得到较硬负载特性的同时需增大Kp和ki，则增大了启动电流.
